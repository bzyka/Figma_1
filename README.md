Прототипирование (Figma) II
Перед вами домашнее задание по теме «Прототипирование (Figma) II».

Цель задания: закрепление полученных знаний.

Для выполнения домашнего задания выберите любой вариант:

Задание (базовый вариант). Подходит для новичков.
Переработайте под Android прототип главного экрана приложения по учёту расходов.
Задание (продвинутый вариант). Если вы знакомы с Figma, выполните задание “со звёздочкой”.
Доработайте экран приложения по учёту расходов, сделав прототип кликабельным.
Ссылку на выполненное задание в Figma загрузите в свой личный кабинет.

Результат
Получите практические знания и создадите под Android прототип главного экрана приложения для учёта расходов.
